# 小剧场生成器 / Theater Generator

独立生成小剧场/番外内容的 SillyTavern 扩展插件。读取当前聊天的剧情上下文和角色设定，在独立的 UI 面板中生成小剧场，不影响正文剧情。

## 功能特性

- **独立生成**：小剧场在插件面板内独立生成和展示，完全不影响正文聊天
- **自由指令**：自由输入任何小剧场主题指令，也可以保存常用指令为模板
- **渲染模板**：内置默认轻量卡片模板，支持自定义渲染规则（可粘贴各位老师的小剧场规则）
- **预设选择**：可跟随当前 API 预设，也可自定义系统提示词
- **历史记录**：保存生成的小剧场，随时回看
- **数据导入导出**：支持导出和导入所有模板和历史数据

## 安装

### 方法一：通过 SillyTavern 扩展管理器安装（推荐）

1. 打开 SillyTavern → Extensions → Install Extension
2. 粘贴本仓库地址
3. 点击安装

### 方法二：手动安装

1. 下载本仓库
2. 将文件夹放入 `data/default-user/extensions/third-party/sillytavern-theater/`
3. 重启 SillyTavern

## 使用方法

1. 在 Extensions 面板中找到「小剧场生成器」
2. 点击「打开小剧场」按钮
3. 在指令标签页输入你想要的小剧场主题
4. 点击生成，等待结果
5. 可以保存到历史或复制 HTML 代码

## 标签页说明

| 标签 | 功能 |
|------|------|
| 指令 | 输入小剧场指令 + 管理指令模板 + 查看生成结果 |
| 模板 | 管理渲染规则模板（小剧场输出格式） |
| 预设 | 选择生成时使用的系统提示词 |
| 历史 | 查看和管理已保存的小剧场 |
| 设置 | 上下文范围、数据导入导出 |

## 兼容性

- SillyTavern 1.12.0+
- 支持所有 API（OpenAI、Claude、Gemini 等）
