# 小剧场生成器 / Theater Generator

独立生成小剧场/番外内容的 SillyTavern 扩展插件。读取当前聊天的剧情上下文、角色设定、世界书和用户人设，在独立面板中生成小剧场，不影响正文剧情。

## 功能

- **独立生成** — 小剧场在插件面板内独立生成，完全不影响正文
- **自由指令** — 自由输入指令 + 保存常用指令为模板
- **渲染模板** — 内置默认模板，支持自定义（可粘贴各位老师的小剧场规则）
- **交互模式** — 可生成带点击/交互功能的小剧场
- **世界书 & 人设** — 读取世界书条目和 User 人设，支持按条目开关
- **独立 API** — 可选择跟随主 API 或配置独立 API 接口
- **历史记录** — 保存生成结果，随时回看
- **自定义 CSS** — 可自定义插件面板样式
- **数据导入导出** — 支持导出和导入所有数据

## 安装

### 方法一：扩展管理器（推荐）

Extensions → Install Extension → 粘贴仓库地址

### 方法二：手动安装

将文件放入 `data/default-user/extensions/third-party/st-theater/`

## 兼容性

- SillyTavern 1.12.0+
- 支持所有 API
